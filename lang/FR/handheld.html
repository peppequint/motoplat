vc-11r

  • 

    Teste tous les alternateurs modernes contrôlés par ordinateur, même les Stop & Start

  • 

    Diagnostique l’ECU, le câblage et l’alternateur à même le véhicule

  • 

    Testeur d’alternateur portatif à même le véhicule

La série VC commence par le testeur d’alternateur « sur le véhicule » VC-11R. Après avoir branché le VC-11R sur l’alternateur, le testeur identifie automatiquement le système de contrôle, et vous pouvez commencer à tester l’alternateur, les commandes envoyées et reçues par l’ECU, et le câblage.

Quelques secondes seulement suffisent au testeur d’alternateur VC-11R pour identifier automatiquement le type de régulateur utilisé sur l’alternateur qui doit être testé (Lin_1, Lin_2, BSS-1,2,3).

Lorsque le testeur vérifie le PWM et la PD, l’écran affiche également des informations sur le signal DFM.

Le réglage initial de la tension du VC-11R est 14,5 V, mais il est possible de le modifier en appuyant sur les boutons « 15,5 V », « 13,5 V », « 12,5 V » ou sur le bouton « OFF ». Cela duplique la situation réelle dans la voiture pendant la charge.

Le signal DFM est affiché en pourcentage (%), ce qui vous permet d’évaluer les performances pendant le chargement à un moment précis.

vc-15w

  • 

    Testeur d’alternateur portatif

  • 

    Teste tous les alternateurs modernes contrôlés par ordinateur, même les Stop & Start

  • 

    Diagnostique les alternateurs qui communiquent avec l’ECU des véhicules

  • 

    Peut être utilisé pour tester l’alternateur à même le véhicule, ou en combinaison avec un banc d’essai conventionnel

Quelques secondes seulement suffisent au testeur d’alternateur VC-15W pour identifier automatiquement le type de régulateur utilisé sur l’alternateur qui doit être testé (Lin_1, Lin_2, BSS-1,2,3). Il est même possible d’identifier le type de régulateur sans l’alternateur... en connectant le B + et B-sur le régulateur, puis en connectant la borne COM à la borne COM du régulateur (S’il vous plaît voir le manuel d’instruction du régulateur pour plus d’informations).

Lorsque le testeur vérifie le PWM et la PD, l’écran affiche également des informations sur le signal DFM. Le réglage initial de la tension du VC-07M est 14,5 V, mais il est possible de le modifier en appuyant sur les boutons « 15,5 V », « 13,5 V », « 12,5 V » ou sur le bouton « OFF ». Cela duplique la situation réelle dans la voiture pendant la charge.

Le signal DFM est affiché en pourcentage (%), ce qui vous permet d’évaluer les performances pendant le chargement à un moment précis.

L’une des fonctions supplémentaires du testeur d’alternateur VC-07M est le voltmètre.
